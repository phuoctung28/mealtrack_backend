# Python Style & Conventions

**Python Version**: 3.11+

## Tools
- **Ruff**: Primary linter and formatter.
- **Mypy**: Strict type checking.

## Standards
- **Line Length**: Max 100 characters (target 88).
- **Quotes**: Double quotes for strings, single quotes for docstrings if needed.
- **Imports**: Standard lib → Third-party → Local.
- **Types**: 100% type hint coverage for all new code.

## Naming
- **Modules/Files**: `snake_case.py`
- **Classes**: `PascalCase`
- **Functions/Variables**: `snake_case`
- **Constants**: `UPPER_SNAKE_CASE`
- **Private**: Prefix with underscore `_name`.
